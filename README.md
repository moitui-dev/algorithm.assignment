## Introduction
This project implements a **Stack** data structure using the C programming language. A stack is a fundamental linear data structure that follows the **Last In, First Out (LIFO)** principle, meaning the last element added to the stack is the first one to be removed. This implementation provides the core stack operations including push, pop, and peek, along with utility functions for stack management.

Stacks are widely used in computer science for various applications such as function call management, expression evaluation, undo operations in editors, and backtracking algorithms. Understanding stack implementation is crucial for mastering data structures and algorithms.

---

## Stack Overview

### What is a Stack?

A stack is an abstract data type that represents a collection of elements with restricted access. Elements can only be added or removed from one end, called the **top** of the stack.
